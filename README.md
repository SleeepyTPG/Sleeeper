# How to use

### - Step 1: Get Python 3.13

### - Step 2: Create venv
```bash
python3.13 -m venv venv
```

### - Step 3: Activate venv
Linux:
```bash
source ./venv/bin/activate
```

Windows:
```bash
idk
```

### - Step 4: Install dependencies
```bash
pip install -r requirements.txt
```

### - Step 5: Set bot token
```bash
cp .env.example .env
```

then add your token

### - Step 6: Start
```bash
./venv/bin/python3.13 -OO main.py
```
